# wcreadme

Hey! This is just where we can put our basic ideas.

Some basics: # does headings, so:
# big
## smaller
### three
#### four
##### five


You can also make bullet points with -
- list
+ list
. list


and numbered lists can be any number, so

1. first thing
2. second thing
3. third thing
    1. sub-thing
